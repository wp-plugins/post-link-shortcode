Link to posts and pages from your content using a simple shortcode.

== Description ==

[Post link shortcode](http://blog.scur.pl/wordpress-post-link-shortcode/) is a WordPress plugin that adds a possiblity to link to posts and pages within your post or page content using a custom shortcode.

For more information, check out [Post link shortcode](http://blog.scur.pl/wordpress-post-link-shortcode/).

Features include:

* Usage as simple as using [post] in your post/page content
* Customising the output by specifying anchor title, text and custom CSS classes

== Installation ==

1. Install Post link shortcode either via the WordPress.org plugin directory, or by uploading the `post-link-shortcode.php` file to your server `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Start using your Post link shortcode! WordPress will automatically recognize [post] shortcode.
